# Personal Website

This repo contains a dependency-free personal website (projects + research + an AI Automation Agency section).

## Quick start

From the repo root:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize content

Edit:

- `assets/content.js` (name, links, projects, research, writing, contact email)

## Deploy

Works on any static host (GitHub Pages, Netlify, Cloudflare Pages, S3, etc). Upload the repository contents as-is.
